# C Programming Refresh
This repository contains my C programming refresh journey.
I started this because I had already studied programming before, but I became rusty and wanted to rebuild my basics properly before moving into Embedded C and ESP32 projects.
The goal of this repository is to revise C concepts step by step through small practical projects.

## What this repository covers
This refresh covers important C programming concepts such as:
* Input and output
* Variables and data types
* If-else conditions
* Loops
* Functions
* Arrays
* Strings
* Pointers
* Structs
* File reading
* File writing
* Menu-based programs
* Basic embedded-style logic

## Why I made this repository
I made this repository to rebuild my C programming foundation in a practical way.
Instead of only revising theory, I created one project for each topic. This helped me understand how C concepts are used in real programs.

## Embedded systems relevance
This refresh is also preparation for Embedded C and ESP32 development.
Many embedded systems work by:
* Reading input values
* Checking conditions
* Storing data
* Controlling output devices
* Logging system events
The later projects in this repository simulate these ideas using normal C programs before moving to real hardware.

## How to run a project
Open any project folder and compile the `main.c` file.
Example:
```bash
gcc main.c -o program
```

Run on Linux or Mac:
```bash
./program
```
Run on Windows PowerShell:
```powershell
./program.exe
```
Each project folder also contains its own README file explaining what the project does, the concepts used, and why it matters.
