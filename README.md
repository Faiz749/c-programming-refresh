C Programming Refresh
This repository contains my C programming refresh journey.
I started this because I already studied programming before, but I became rusty and wanted to rebuild my basics properly.
I will add one project at a time as I revise C concepts like input/output, conditions, loops, arrays, strings, functions, structs, file handling, and pointers

What it does
This C program takes obtained marks and total marks, calculates percentage, prints grade, and shows pass/fail status.

 Concepts used
- printf
- scanf
- if else
- validation
- percentage calculation
- float type

How To Run
On Windows, you can run:
grade-calculator.exe

What I Learned
From this project, I learned how to take user input, validate it, calculate percentage, and use if-else conditions to make decisions in a program.
